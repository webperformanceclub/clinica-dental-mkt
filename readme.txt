# Landing comercial WebPC para clínicas dentales

## Objetivo

Crear una landing page comercial para WebPC orientada a vender servicios web a clínicas dentales locales.

La página no debe parecer una web técnica ni una simple plantilla. Debe transmitir que WebPC ayuda a clínicas dentales a conseguir más primeras visitas, mejorar su presencia online y facilitar que los pacientes pidan cita desde el móvil.

La página debe estar enfocada a conversión, con llamadas claras a la acción: solicitar demo, pedir valoración gratuita o contactar por WhatsApp.

---

## Stack recomendado

- Next.js con App Router
- TypeScript
- Tailwind CSS
- Despliegue en Vercel
- Diseño responsive mobile-first
- Página rápida, sencilla y sin dependencias innecesarias

## Mi recomendación para vender mejor esta página

Yo haría la landing con un enfoque muy directo:

**No vender “diseño web”.**  
Vender:

> “Una web preparada para que una persona que busca dentista desde el móvil pueda confiar y pedir cita rápido.”

Ese argumento es mucho más comercial.

También pondría una oferta de entrada clara:

> **Landing dental profesional desde 490 €**  
> Ideal para clínicas que quieren mejorar su presencia online sin hacer una inversión grande.

Y luego empujaría al paquete recomendado:

> **Pack Captación desde 890 €**  
> Para clínicas que quieren una web más enfocada a conseguir primeras visitas.

Esa estructura os permite entrar con precio razonable, pero no quedaros atrapados en proyectos baratos.
